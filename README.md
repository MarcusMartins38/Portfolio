### How Can I use ?

- With Yarn
```
$ git clone https://github.com/MarcusMartins38/Portfolio.git
$ cd Portfolio
$ yarn
$ yarn start
```
- With Npm

```
$ git clone https://github.com/MarcusMartins38/Portfolio.git
$ cd Portfolio
$ npm install
$ npm start
```
